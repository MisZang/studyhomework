# 简单学生信息管理系统

## 项目概述

简单学生信息管理系统是一个基于 `Tkinter` 的桌面应用程序，用于管理学生信息。该系统支持添加、查看、修改和删除学生信息，并提供导入和导出 Excel 文件的功能。

## 功能列表

- 添加学生信息
- 查看学生信息
- 修改学生信息
- 删除学生信息
- 导入/导出 Excel 文件
- 滚动查看学生信息
- 自动加载最近的 Excel 文件

## 安装与运行

### 安装依赖

在运行 `main.py` 前，确保已安装 Python 环境，程序会自动检查并安装以下库：

- `tkinter`
- `pandas`
- `matplotlib`

### 运行程序

在终端或命令行中运行以下命令：
